计算机网络笔记

发表于 2021-10-24 更新于 2021-10-30 分类于 工作学习 本文字数： 2.7k 阅读时长 ≈ 2 分钟

大二暑假计算机网络学习笔记总结 为了408 冲冲冲！

综述

计算机网络：

  • 概念：计算机网络是互联的、自治的计算机集合
  • 组成：
    • 组成上：硬件、软件、协议
    • 工作方式上：
      • 边缘部分：客户端
      • 核心部分：服务器、路由
    • 功能组成上：通信子网、资源子网
  • 功能
    • 数据通信
    • 资源共享
    • 分布式处理：一个任务交给多台计算机
    • 提高可靠性：计算机的备胎
    • 负载均衡：均摊任务
  • 分类
    • 分布范围：广域网、城域网、局域网、个人区域网
    • 使用者：公用网、专用网
    • 交换技术：电路交换、报文交换、分组交换
    • 拓扑结构：总线型、星型、环形、网状型
    • 传输技术：广播式、点对点
  • 标准化工作及相关组织
    • 标准：
      • 法定标准
      • 事实标准
    • RFC及其发展为因特网标准的四个阶段(补充):
      • Request for Comment:
        • 1.因特尔草案
        • 建议标准(网友评价)
        • IETF、IAB审核
        • 通过
    • 相关组织：ISO、ITU、IEEE、IETF
  • 性能指标
  • 体系结构与参考模型
    • OSI参考模型
    • TCP/IP模型
    • 五层参考模型

基础概念

  • 性能指标：
    • 速率：数据在发射端的发送速度
    • 带宽：车道宽度
    • 吞吐量：单位时间内通过的数据(b/s Kb/s Mb/s)
  • 时延:
    • 发送时延
    • 传播时延
    • 排队时延(等待处理)
    • 处理时延
  • 时延带宽积： 每秒发送的数据(b Kb Mb)
  • RTT(往返时延)：数据开始传播 -> 接到并处理完毕 的时间
  • 利用率：
    • 信道利用率：有数据通过的时间/总时间
    • 网络利用率：信道利用率加权平均值

相关概念：

  • 对等实体(同一层)：遵循同一协议
  • 下层为上层通过 接口 提供 服务
  • 协议包括：
    • 语法：规定传输数据的格式
    • 语义：规定所要完成的功能
    • 同步：规定各种操作的顺序

PCI：协议控制信息

SUD：服务数据单元：垂直提供服务的数据

PDU：协议数据单元：水平对等协议的传送的数据单位

OSI参考模型

  • 物 ->物理层

    • 定义接口特性
    • 定义传输模式 -> 单工、半双工、双工
    • 定义传输速率
    • 比特同步

  • 联 ->数据链路层

    • 成帧

    • 差错控制

    • 流量控制

    • 访问接入控制

    • 主要协议：SDLC、HDLC、PPP、STP

      帧

  • 网 ->网络层

    • 路由选择 ->最佳路径
    • 流量控制
    • 差错控制
    • 拥塞控制

​ 主要协议：IP、IPX、ICMP、IGMP、ARP、RAFP、DSPF

​ 分组（单位：数据报）

  • 淑 ->传输层

    • 可靠（确认后传输，大数据）、不可靠传输（小数据）

    • 差错控制

    • 流量控制（控制速度）

    • 复用（数据一起打包）、分用（根据端口拆包）

      数据

  • 慧 ->会话层

    • 建立、管理、终止会话
    • 使用校验点 实现数据同步

  • 试 ->表示层

    • 数据格式变换
    • 数据加密解密
    • 数据压缩恢复

  • 用 ->应用层

    • 文件传输（FTP）
    • 电子邮件（SMTP）
    • 万维网（HTTP）

TCP/IP参考模型&五层参考模型

  • OSI参考模型
    • 应用层
    • 表示层
    • 会话层
    • 传输层
    • 网络层
    • 数据链路层
    • 物理层
  • TCP/IP参考模型
    • 应用层(应用层、表示层、会话层)
    • 传输层
    • 网际层
    • 网络接口层(数据链路层、物理层)
  • TCP/IP协议栈
    • 应用层：HTTP、FTP、DNS
    • 传输层：TCP、UDP
    • 网际层：IP
    • 网络接口层：Ethernet、ATM、Frame Relay
  • 五层参考模型：
    • 应用层 -> 支出各种网络应用 FTP、SMTP、HTTP
    • 传输层 -> 进程-进程的数据传输 TCP、UDP
    • 网络层 -> 源主机到目的主机的数据分组路由与转发 IP、ICMP、OSPF等
    • 数据链路层 -> 把网络层传下来的数据报组装成帧 Ethernet、PPP
    • 物理层 -> 比特传输

面向连接：第一是建立连接，在此阶段，发出一个建立连接请求。只有在连接建立成功之后，才能开始数据传输，这是第二阶段。接着，当数据传输完毕，必须释放连接。

无连接：直接进行数据传输

ISO/OSI参考模型 TCP/IP模型
网络层 无连接+面向连接 无连接
传输层 面向连接 无连接+面向连接

Attention:

  • 1.OSI定义三点：服务、协议、接口
  • 2.参考模型先于协议
  • 3.TCP/IP设计之初就考虑到异构网互联问题，将IP作为重层次

数据通信基础

  • 物理层：

    • 概念

    • 四种特性

      • 机械特性: 物理连接所采用的规格、接口形状、引脚数目等
      • 电气特性: 线路上信号的电压范围、阻抗匹配、传输速率等
      • 功能特性: 某一电平的意义
      • 规程特性(过程特性): 各条物理线路的工作规程和时序关系

    • 两个定理

      • 奈氏准则
      • 香农定理

    • 物理层传输介质

      • 导向: 双绞线、同轴电缆、光纤
      • 非导向
        • 无线电波 -> 所有方向 -> 手机通信
        • 微波 -> 单一方向 -> 卫星通信
        • 红外线、激光 -> 单一方向

  • 物理层设备

    • 中继器: 对信号还原和再生
    • 集线器: 对信号放大转发(冲突域)

  • 术语

    • 码元、速率、波特、带宽
      • 码元: 固定时长的信号波形 -> 有具体意义(可携带多比特的信息量)
      • 码元传输速率: 1s传输多少个码元 (单位：Baud波特)
      • 信息传输速率: 1s传输多少个比特
      • 宽带: 最高数据率 (天花板)
    • 数据、信号、信道、信源、信宿

  • 三种通信方式

    • 单工
    • 半双工
    • 全双工

  • 两种数据传输方式

    • 串行
    • 并行

  • 编码与调制

    • 信号

      • 数字与模拟
      • 基带与宽带
        • 基带信号：原生信号(信源) -> 弱
        • 宽带信号：调制后的高频信号 -> 强

    • 编码

      • 数字数据编码为数字信号

        • 1.归零编码：后一半归零
        • 2.非归零编码：后一半归一
        • 3.曼彻色特编码：指定先高后低 ->1 / 先低后高 -> 1
        • 4.差分曼彻色特编码：一半 -> 同1异0 （跳变）
        • 5.反向不归零编码：同1异0（跳变）
        • 6.4B/5B编码：4/5 编码效率：80%

      • 模拟数据编码为数字信号

        • PCM编码器

    • 调制

      • 数字数据调制为模拟信号
      • 模拟数据调制为模拟信号

数据链路层

  • 概述和功能

    • 概述: （搬运工）将网络层发送给它的数据报，安全无差错的传给它的相邻节点 -> 1.加强物理层传输原始比特流的功能 2.将物理层提供的可能出错的物理连接改造成逻辑上无差错的数据链路
    • 功能
      • 为网络层提供服务 -> 1.有确定无连接服务 2.无确定无连接服务 3.有确定面向连接服务（有连接一定有确定）
      • 链路管理 ->建立、维持、释放 -> 面向连接的服务
      • 组帧
      • 流量控制
      • 差错控制（位错/帧错）

  • 封装成帧和透明传输

    • 字符计数法

    • 字符(节)填充法

    • 零比特填充法: 5 '1' 1 '0' 出现五个1 立即填充一个0

    • 违规编码法

    (由于字节填充法Count字段的脆弱性（其值发生改变将造成灾难性的后果），因此大都使用后两种 -> 2.零比特填充法 3.违规编码法)

  • 流量控制和可靠传输

    • 停止等待协议
    • GBN
    • SR

  • 差错控制

    • 检错
    • 纠错

  • 介质访问控制

    • 静态划分信道
    • 动态分配信道
      • 随机访问介质控制
      • 轮询访问介质控制

  • 局域网、以太网802.3、无线局域网802.11

  • 广域网使用的链路层协议

    • PPP协议
    • HDLC协议

  • 链路层设备

    • 网桥
    • 交换机
    • 冲突域和广播域
